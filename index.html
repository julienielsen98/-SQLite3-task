NORWEGIAN COSTUMERS OVERVIEW

USAGE: to view...

Location:

write "/location/:store_location" in the URL

example: localhost:8081/location/Porsgrunn, Telemark

Make sure to use comma and space between city and state, and capital letter


Store:

write "/store/:store_name" in the URL

eksample: localhost:8081/store/Rema1000

Make sure to use capital letter


Costumer/User:

write "/card/:card_number" in the URL

example: localhost:8081/card/1897


All data for given date:

write "/day/:date" in the URL

example: localhost:8081/day/14


All data for given month:

write "/month/:month_number/:year_number" in the URL

example: localhost:8081/month/11/22



View all users:

write "/cards" in the URL

example: localhost:8081/cards


View all items:

write "/items" in the URL

example: localhost:8081/items

View all data:

write "/showall”" in the URL

example: “localhost:8081/showall”
